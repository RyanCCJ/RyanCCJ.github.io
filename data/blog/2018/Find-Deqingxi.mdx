> 德慶溪又稱坑仔底，是一條發源於臺南臺地、流經臺南市區的河流。由於都市計畫，地貌發生劇烈的改變，河道已從地面上消失。
> 我們希望重新構築溪流與周遭地景之間的關係，並結合歷史資料或是人物故事，進而建構起在當時生活的景況。

## 專案介紹

**發現德慶溪 (Find Deqingxi)** 是大學時期的網頁專案，由四個成大電機系與三個工業設計系同學所組成的團體，目標是建構一個環繞在德慶溪流域的導覽解說平台。
這個專案主要使用 web app 技術，讓它能夠在電腦與手機中跨平台運作。前端使用 jQuery，後端使用 PHP 搭配 MongoDB，並以 Apache 作為伺服器部署此網站。

發現德慶溪包含導覽者社群平台，讓在地導覽員能夠建立自己的口碑、評論與紀錄，一般用戶則可以透過此平台尋找合適的導覽員。
![](/static/images/2018/Find-Deqingxi/navigator.png)

另外，此平台附有導覽地圖，透過 Google Map API 使其能與文獻圖資做疊圖，並標記幾個重要導覽地點供用戶與導覽員作參考，
包含臺南一中、臺南廳長官邸、吳園、臺南公會堂、禾寮港、民族戲院、大觀音亭、開基天后祖廟、鴨母寮市場、拱乾門、三山國王廟等特色景點。
![](/static/images/2018/Find-Deqingxi/map.png)

用戶可透過 Facebook、Google、Twitter 等社群軟體登入該平台，並與當前活躍的導覽員做匹配。
一旦匹配成功，用戶便可以漫步在台南府城過去的德慶溪流域，透過線上語音即時得到導覽員的解說，並於導覽結束後留下心得和評論。

![](/static/images/2018/Find-Deqingxi/templates.jpg)

## 專案回顧

在本專案中，我主要負責兩個項目，包含前端頁面佈局與社群帳號登入的功能。
頁面佈局的部分像是整體使用流程、Sidebar、click 等 JavaScript 事件。
社群登入功能例如透過 Facebook SDK 獲取用戶 ID、名稱與 access token，並將社群頭貼當作帳戶頭貼使用。

這是一個為期數個月的網頁開發課程專案，電機系與工設系彼此分工合作，由他們提出設計、我們實作細節，並由 leader 總結方向。
在這過程中，我們從零開始學習新技術，完成一個完整的專案，並透過與不同領域的人合作，激盪出更多創意。
儘管團隊最終並沒有完成所有預期的目標，但也成功實作出登入平台與地圖導覽功能，並成功參與學期末對外的公開展覽，是難得的體驗。
